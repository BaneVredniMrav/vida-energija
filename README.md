# Vida Energija - Web sajt

Web sajt za elektroinstalaterske usluge izgrađen sa Vue 3 + Vite + Tailwind CSS.

## Tehnologije

- **Vue 3** - JavaScript framework sa Composition API
- **Vite** - Build alat za brz razvoj
- **Tailwind CSS** - Utility-first CSS framework

## Paleta boja

Primarna zelena boja iz logoa je `#22c55e` (vida-500).

## Instalacija

```bash
npm install
```

## Razvoj

```bash
npm run dev
```

## Build za produkciju

```bash
npm run build
```

## Pregled builda

```bash
npm run preview
```
